# Product Filtering System

A responsive product catalog application built with HTML, CSS, and JavaScript. The application dynamically loads products from a local JSON file and provides real-time search, category filtering, and price sorting functionality.

## Features

- Load products from a local JSON file
- Search products by name
- Filter products by category
- Sort products by price
- Display filtered results count
- Responsive grid layout
- Dynamic DOM rendering
- Optimized updates using DocumentFragment
- Clean separation of business logic and UI logic

## Technologies Used

- HTML
- CSS
- JavaScript
- JSON

## How It Works

1. Products are loaded from `products.json`.
2. Users can search products by name.
3. Products can be filtered by category.
4. Products can be sorted by price.
5. Results update instantly without reloading the page.
6. Product cards are rendered dynamically using JavaScript.
